<h2>💌 Sobre o projeto</h2>
<p>
Nosso Cantinho foi o meu <strong>primeiro site</strong>, criado do zero como presente para a minha namorada.
</p>
<p>
O projeto foi desenvolvido com HTML, CSS e JavaScript puro — sem frameworks, sem bibliotecas. Tudo na mão, do jeito mais raiz possível. Mais do que um exercício técnico, foi um projeto com significado real, o que tornou o processo de aprendizado ainda mais especial.
</p>

<h2>✨ Funcionalidades</h2>
<p>✅ Página personalizada com fotos e memórias</p>
<p>✅ Design criado do zero com CSS puro</p>
<p>✅ Interatividade com JavaScript</p>
<p>✅ Feito com muito carinho 💕</p>
